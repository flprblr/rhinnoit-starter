# Favicon Directory

This directory contains all favicon-related files for the application.

## Required Files

- `favicon.ico` - Standard favicon (48x48px recommended)
- `favicon.svg` - Modern SVG favicon (scalable, preferred by modern browsers)
- `apple-touch-icon.png` - Apple touch icon (180x180px)
- `site.webmanifest` - Web app manifest file

## Recommended Additional Files

### Essential PNG Sizes:
- `favicon-16x16.png` - For older browsers
- `favicon-32x32.png` - Standard modern browsers
- `favicon-96x96.png` - Android devices

### Android Chrome (PWA):
- `android-chrome-192x192.png` - Standard Android icon
- `android-chrome-512x512.png` - High-resolution Android icon

### Optional:
- `favicon-180x180.png` - Alternative iOS icon
- `mstile-150x150.png` - Windows tiles (if supporting Windows)

## Generation Tips

1. Start with a high-resolution source (at least 512x512px)
2. Generate all sizes from the same source for consistency
3. Use PNG format for raster images
4. Keep SVG simple and optimized
5. Test favicons across different browsers and devices
