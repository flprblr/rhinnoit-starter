# SEO Images Directory

This directory contains images used for SEO and social media sharing.

## Required Files

### PNG Format (Fallback):
- `og-image.png` - Open Graph image for Facebook/LinkedIn (1200x630px recommended)
- `twitter-image.png` - Twitter Card image (1200x675px recommended)

### WebP Format (Recommended for 2025):
- `og-image.webp` - Optimized Open Graph image (1200x630px)
- `twitter-image.webp` - Optimized Twitter Card image (1200x675px)

## Recommended Additional Sizes

### Square Format:
- `og-image-square.png` / `og-image-square.webp` - Square image (1200x1200px) for LinkedIn and other platforms

### Mobile Optimized:
- `og-image-mobile.png` / `og-image-mobile.webp` - Mobile version (600x315px) for faster loading

## Image Guidelines

### Dimensions:
- **OG Image**: 1200x630px (1.91:1 ratio) - Facebook, LinkedIn
- **Twitter Image**: 1200x675px (16:9 ratio) - Twitter/X
- **Square Image**: 1200x1200px (1:1 ratio) - LinkedIn, Instagram

### Best Practices:
- Use high-quality images that represent your brand/application
- Include text/logo if appropriate
- Keep file sizes reasonable for fast loading (< 1MB recommended)
- Use WebP format with PNG fallback for better compression
- Ensure text is readable at small sizes
- Test images on different platforms before publishing

### Format Priority:
1. **WebP** - Modern format with better compression (use when possible)
2. **PNG** - Fallback for older browsers
3. Always provide both formats for maximum compatibility
